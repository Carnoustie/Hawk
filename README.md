# Centralbank
Bond market coverage app.<br><br>After recently (December 2025) reading some history books, I have gotten interested in following the sovereign bond and currency markets, due to their influence on daily life. I thus decided to build a web app where the user can eaily follow those markets, as well as monetary policy events.<br><br>Minimal usage of AI tools to preserve control of code base and optimize learning.
